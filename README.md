# UzFlora Frontend

Bu repo UzFlora loyihasi uchun frontend ilovaning boshlang'ich skaffolddir (React + Vite).

Tez boshlash:

1. Klonlash:
   git clone https://github.com/BoburChoriyev0799/uzflora-frontend.git
   cd uzflora-frontend
2. Paketlarni o'rnatish:
   npm install
3. Dev serverni ishga tushirish:
   npm run dev
4. Build:
   npm run build

Deploy:
- GitHub Actions orqali build qilingan distributsiya `gh-pages` branchga joylanadi. GitHub Pages sozlamalarida `gh-pages` branchni tanlang yoki Vercel/Netlify bilan deploy qiling.
